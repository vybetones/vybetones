# 🎸 Fretboard Trainer — Изучи гриф гитары

Интерактивное веб-приложение для изучения нот на грифе гитары, оптимизированное под мобильные устройства.

## ✨ Возможности

### Режимы обучения

- **🎸 Обзор** — свободное исследование грифа. Тапай по нотам, чтобы увидеть их названия, октавы и интервалы.
- **🎵 Гаммы** — визуализация гамм на грифе. Выбирай тональность и тип гаммы (Major, Minor, Pentatonic, Blues, Dorian, Mixolydian, Harmonic Minor).
- **🎶 Аккорды** — изучение аккордовых форм. Поддержка Major, Minor, Dom7, Maj7, m7, Sus2, Sus4, Dim, Aug.
- **🔍 Найди ноту** — тренажёр: приложение называет ноту, ты находишь её на грифе.
- **❓ Угадай ноту** — тренажёр: приложение показывает позицию, ты угадываешь ноту.

### Функции

- 🎯 Интерактивный гриф с 12-24 ладами
- 🎵 Звуковая обратная связь при правильных/неправильных ответах
- 📊 Статистика прогресса: точность, серии, история тренировок
- 🎨 Тёмная тема по умолчанию
- 📱 Mobile-first дизайн с крупными кнопками
- 💾 Локальное сохранение прогресса (localStorage)
- ⚙️ Настройки: выбор строя, количества ладов, ориентации (леворукий режим)
- 📴 PWA поддержка — работает офлайн
- 🔥 Подсчёт серий правильных ответов

### Музыкальная логика

Все расчёты нот, интервалов, гамм и аккордов вынесены в отдельные модули:

- **notes.ts** — базовая теория: хроматический ряд, интервалы, октавы, позиции нот на грифе
- **scales.ts** — определения гамм (интервалы от корня), проверка вхождения ноты в гамму
- **chords.ts** — определения аккордов, роли нот в аккорде
- **fretboard.ts** — генерация полного грифа с учётом выбранной тональности/гаммы/аккорда

### Расширяемость

Архитектура позволяет легко добавлять:
- Новые типы гамм (добавить в `SCALE_DEFINITIONS`)
- Новые типы аккордов (добавить в `CHORD_DEFINITIONS`)
- Новые режимы тренировки (создать компонент и добавить в `AppMode`)
- Альтернативные строи (добавить в `storage.ts`)

## 🎮 Использование

### Режим "Обзор"
1. Выбери тональность (например, C)
2. Переключайся между гаммами и аккордами
3. Тапай по нотам на грифе для получения информации

### Режим "Найди ноту"
1. Приложение показывает ноту (например, "G")
2. Найди и тапни эту ноту на грифе
3. Получи мгновенную обратную связь

### Режим "Угадай ноту"
1. Приложение показывает позицию (струна + лад)
2. Выбери правильное название ноты из вариантов
3. Тренируй реакцию и запоминание

## ⚙️ Настройки

- **Строй гитары**: Standard (EADGBE), Drop D, Half Step Down
- **Количество ладов**: 12, 15, 22, 24
- **Леворукий режим**: зеркальное отображение грифа
- **Октавы**: показывать номера октав вместе с нотами
- **Интервалы**: показывать интервалы относительно корня
- **Звуки**: включить/выключить звуковую обратную связь
